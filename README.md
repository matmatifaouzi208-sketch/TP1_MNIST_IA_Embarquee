# TP1_MNIST_IA_Embarquee

## Introduction :

[cite_start]Ce projet s'inscrit dans le cadre du module ETRS606 (IA Embarquée) à l'Université Savoie Mont Blanc[cite: 1, 2, 3]. [cite_start]Il a pour objectif de résoudre le problème MNIST[cite: 3].
* [cite_start]Le jeu de données MNIST est un benchmark classique en apprentissage automatique pour la classification d'images[cite: 4].
* [cite_start]Il contient des images en niveaux de gris de chiffres manuscrits (0 à 9)[cite: 5].
* [cite_start]L'objectif est de prédire correctement la classe (le chiffre) associée à chaque image parmi les 10 classes possibles (0-9)[cite: 7].
* [cite_start]Chaque image est de taille 28x28 pixels, soit 784 pixels au total[cite: 6]. 
* [cite_start]Pour l'entraînement, chaque image [28x28] est transformée en un vecteur aplati [1x784][cite: 14].
